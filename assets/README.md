# Assets

Static resources and branding materials for the KaryoExplorer web application.

## Directory Structure

```
assets/
├── logo/                          # Customizable logos for web interface branding
│   ├── left_icon.png              # Left-side logo/icon (100x100px recommended)
│   └── right_icon.png             # Right-side logo/icon (100x100px recommended)
└── figures/                       # Documentation figures and screenshots
    ├── chrosome_view.gif          # Chromosome view visualization example
    ├── home_page.gif              # Home page interface demonstration
    ├── Home_Page.png              # Home page static screenshot
    ├── karyogram.gif              # Karyogram visualization example
    ├── pre_post_diff_karyograms.png  # Pre/post differential karyogram comparison
    └── pre_post_LRR_BAF.png       # Pre/post LRR/BAF plot comparison
```

## Usage

### Logo Customization

Replace the placeholder logo files in `assets/logo/` with your institutional branding:
- Maintain PNG format
- Keep the same filenames: `left_icon.png` and `right_icon.png`
- Recommended dimensions: 100x100 pixels (square format)


### Figures

The `figures/` directory contains visual documentation used in README and documentation files to demonstrate pipeline capabilities and outputs.
