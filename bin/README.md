# Pipeline Scripts

Core analysis scripts and tools used by the KaryoPlayground pipeline.

## Directory Structure

```
bin/
├── analysis/         # IBD, LOH, and ROH analysis scripts
├── cnv/              # Copy number variation analysis and summarization
├── dynamic_plotting/ # Interactive web application generation
├── functions/        # Shared R functions and utilities
├── genotyping/       # Genotype matching and comparison
├── plotting/         # Static visualization scripts (LRR-BAF plots)
└── qc/               # Quality control and data preprocessing
```
